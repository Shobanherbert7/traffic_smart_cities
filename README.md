# traffic_smart_cities
AI- Powered urban traffic analysis system using LSTM, ARIMA &amp; clustering to predict congestion, detect hotspots , and support smart city decision-making in real time. 



Urban traffic congestion costs cities billions annually in lost productivity and emissions. This project tackles that problem with a full ML pipeline — from raw IoT sensor and GPS data ingestion, through time series forecasting (LSTM + ARIMA), clustering-based hotspot detection (K-Means, DBSCAN), to an interactive Plotly dashboard for city planners and traffic authorities.

Built as part of a smart city AI initiative, it demonstrates real-world application of predictive analytics, geospatial reasoning, and decision-support tooling on large-scale temporal data.

## Features
- Time series forecasting with LSTM networks and ARIMA models
- Congestion hotspot detection using K-Means and DBSCAN clustering
- Accident risk prediction based on flow anomalies and historical incidents
- Weather & event calendar integration for external factor modeling
- Interactive Plotly/Folium dashboard with live congestion heatmaps
- Dynamic signal optimization decision-support module
- Automated city planner reports with infrastructure recommendations
